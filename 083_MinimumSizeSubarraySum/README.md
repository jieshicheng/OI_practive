LeetCode 209 Minmum Sisze Subarray Sum

Answer:
    线性复杂度解决这个问题。题目要求的是求大于给定数值s的最小连续子数组长度。
    我们在顺序遍历整个数组，持续累加和。记录最小的长度ans，以及当前累加和的数组长度length
    当当前累加和数组的值大于s，则我们找到一个解，于是我们另
    ans = min(ans, length)。这样ans保存最小的解。
    其次我们当前累加和已经满足条件了，那么我们可以尝试删除一点多余的值，由于要求是连续数组
    肯定不能删中间的，尾部数据是我们刚加进来的也不用删除，那么我们可以尝试删除头部的值，使得length减少
    这样一直删除头部的值，直到当前累加和数组小于s，我嫩再继续遍历数组
