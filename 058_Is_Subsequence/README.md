问题：
    给定字符串s和t，检查s是否是t的子串
    你可以假定s很短，t很长。
    子串的定义是s的每个元素都在t中出现，并且顺序一致
    例如：
    “abc”，“aseqwebsadc”
    true

思路：
    因为这里t很大，所以朴素的匹配方法可能会有很大的复杂度
    我们遍历t用map保存t中元素出现的位置，然后遍历s，寻找s的每个元素是否出现在map里面并且顺序是可取的。
    比如“absaba” => map[a] = {0,3,5}
                    map[b] = {1,4}
                    map[s] = {2}

    然后遍历“abs”    'a' = 0
                     'b' = 1
                     'c' = 2 => true

    如果存在某个元素找不到下标=> false
